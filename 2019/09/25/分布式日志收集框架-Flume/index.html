Flume 概述

官网

FLume 是有 Cloudera 提供的一个分布式，高可靠，高可用的服务，用于分布式的海量日志的高效收集、聚合、移动系统

设计目标：可靠性，扩展性，管理性

安装

  • 安装 JDK

  • 下载 flume 安装包

  • 解压安装包

  • 更改配置文件

    修改 conf 下的 flume-env.sh.template 复制一份到 flume-env.sh 并修改文件中的 JAVA_HOME 为 jdk 的真实路径

Flume 架构及核心组件

  • Source：收集

    指定数据的来源

  • Channel：聚集

    为数据提供一个临时缓存的地方

  • Sink：输出

    从 Channel 中将数据读取出来，输出到指定位置

Flume 配置文件描述

  • a1：agent 名称
  • r1：source 的名称
  • k1：sink 的名称
  • c1：channle 的名称
